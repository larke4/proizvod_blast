## 3. Постановка задачи

### Цель
Реализовать функционал **TRMM** (Triangular Matrix Multiply) из BLAS уровня 3.

### Операция
```
B = α × op(A) × B   (когда Side = Left)
B = α × B × op(A)   (когда Side = Right)
```

где:
- `A` — треугольная матрица (верхняя или нижняя)
- `B` — прямоугольная матрица
- `op(A)` = A или Aᵀ (транспонированная)
- `α` — скалярный множитель

### Требования
- Поддержка одинарной (`float`) и двойной (`double`) точности
- Совместимость с CBLAS API
- Покрытие всех комбинаций параметров
- Тестирование корректности (минимум 10 тестов)
- Бенчмарк производительности (10 запусков, среднее геометрическое)

## 4. Теоретическое описание TRMM

### Параметры функции

| Параметр | Значения | Описание |
|----------|----------|----------|
| `Order` | `CblasRowMajor`, `CblasColMajor` | Порядок хранения матриц |
| `Side` | `CblasLeft`, `CblasRight` | Сторона умножения |
| `Uplo` | `CblasUpper`, `CblasLower` | Тип треугольной матрицы |
| `TransA` | `CblasNoTrans`, `CblasTrans` | Транспонирование A |
| `Diag` | `CblasNonUnit`, `CblasUnit` | Диагональ (обычная/единичная) |
| `M, N` | int | Размеры матрицы B (M × N) |
| `alpha` | float/double | Скалярный множитель |
| `A, lda` | ptr, int | Треугольная матрица и шаг строки |
| `B, ldb` | ptr, int | Матрица B и шаг строки |

### Сложность алгоритма
- Временная сложность: **O(M × N × min(M, N))**
- Для квадратной матрицы: **O(N³)**
- Память: **O(M × N)** дополнительная

### A. Инструкция по сборке

```bash
# Компиляция тестов
gcc -o test_strmm.exe test_strmm.c strmm.c -lm
gcc -o test_extended.exe test_extended.c strmm.c -lm

# Компиляция бенчмарка
gcc -O2 -o bench_compare.exe bench_compare.c strmm.c my_blas.c -lm

# Компиляция примера
gcc -o example.exe example.c strmm.c -lm
```

### B. Пример использования

```c
#include "strmm.h"

// B = A * B (A — нижняя треугольная 4×4)
float A[16] = {
    2, 0, 0, 0,
    1, 3, 0, 0,
    4, 2, 5, 0,
    1, 1, 1, 2
};
float B[8] = {1, 2, 3, 4, 5, 6, 7, 8};

cblas_strmm(CblasRowMajor, CblasLeft, CblasLower,
            CblasNoTrans, CblasNonUnit,
            4, 2, 1.0f, A, 4, B, 2);
```
